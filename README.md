# ECHOWALL

An interactive emotional wall made with code, motion, sound, and Diet Coke inspired pixel art.

This project started from one random Pinterest rabbit hole.

Those Diet Coke can heart wall arts just looked insanely aesthetic. Minimal wall, soft shadows, silver cans, pixelated heart shape. It felt nostalgic, soft, futuristic, and weirdly emotional at the same time.

So instead of recreating it as static wall decor, the idea became:

what if the wall actually reacted to you?

What if the cans moved with music?
What if hovering over the wall created ripples?
What if the heart could pulse, morph, glow, or breathe?
What if frontend could feel more like an interactive installation instead of just another website?

That’s basically what ECHOWALL is.

Not a SaaS.
Not a productivity app.
Not another finance dashboard.

More like a digital emotional object built with frontend, UI/UX, motion, and atmosphere.

## The Diet Coke Heart

The heart is not an image.

Every single can is generated through code.

Each can is individually rendered, positioned, animated, and controlled using a matrix system.

Basically:

* `1` means visible can
* `0` means empty space

That matrix creates the heart shape.

So instead of placing one giant image, the heart is built can-by-can.

That also means we can:

* animate every can separately
* make the heart react to sound
* create ripple interactions
* morph the shape into something else
* add motion physics
* create pulse effects
* generate particles and lighting dynamically

The cans themselves are inspired by Diet Coke aesthetics without directly copying the branding.

We’re recreating the vibe:

* silver metallic body
* soft reflections
* glossy highlights
* red typography
* minimal premium feel

The goal is making them feel physical and mounted on the wall instead of floating UI elements.

## Modes We’re Building

ECHOWALL will have multiple visual modes.

Minimal mode:
soft gallery lighting, calm shadows, neutral atmosphere.

Japanese mode:
warmer tones, quieter visuals, zen-inspired feeling.

Cyber mode:
darker environment, stronger glow, neon reflections, reactive motion.

Dreamy mode:
soft bloom, floating particles, pastel ambient lighting.

Arcade mode:
retro-inspired pixel energy with stronger reactions and movement.

The cool part is the same heart system powers all modes. Only the atmosphere changes.

## Interaction Systems

This project is heavily interaction-focused.

The wall is supposed to feel alive.

### Mouse React

Hovering across the wall creates:

* ripple movement
* nearby can motion
* subtle depth shifting
* magnetic interactions

Almost like the wall notices your presence.

---

### Music Reactive Mode

Using audio analysis, the heart reacts to music in real time.

Things like:

* pulse animations
* beat-based scaling
* wave movement
* light intensity shifts
* ambient vibration

The goal is making the wall feel emotional instead of just reactive.

---

### Microphone Input

Speaking into the mic changes the wall behavior.

Voice volume and frequencies can:

* expand the heart
* distort shapes
* create ripple waves
* affect glow intensity

---

### Morph Transitions

The heart will eventually morph into:

* stars
* smileys
* clouds
* abstract forms
* glitch patterns

All generated dynamically using the same matrix system.

---

### Atmosphere & Particle FX

Small atmospheric systems will constantly make the scene feel alive.

Things like:

* floating particles
* cinematic grain
* moving shadows
* bloom glow
* subtle reflections
* depth haze
* ambient motion

The goal is “calm immersive frontend”.
